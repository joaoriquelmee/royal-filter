# 👑 Royal Filter

Ferramenta de produtividade desenvolvida para otimizar a gestão da agenda do **Royal Tênis Clube** no sistema Letzplay.

## 🎾 Sobre

O Royal Filter adiciona um painel visual interativo sobre a agenda do dia, permitindo filtrar e analisar as turmas de forma rápida e eficiente, sem precisar navegar pelo sistema manualmente.

## ✅ Funcionalidades

- **Turmas com vagas** — filtra e exibe apenas as turmas que ainda possuem vagas fixas ou vagas disponíveis no dia
- **Aulas com experimentais** — identifica turmas que possuem alunos em aula teste
- **Horários para locação** — exibe os horários disponíveis para aluguel de quadras
- **Filtro por professor** — selecione um ou mais professores para visualizar apenas as suas turmas do dia
- **Relatório de presença** — gera automaticamente um relatório formatado por turno (manhã/tarde) com os professores de cada horário, pronto para copiar e enviar no WhatsApp
- **Resumo do dia** — contador com total de turmas com vaga, turmas lotadas, alunos experimentais e horários disponíveis para locação
